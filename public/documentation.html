Favorite Movies API

This API provides access to information about favorite movies, directors, and plots. Users can sign up, update their personal information, and create a list of their favorite movies.

API Endpoints

1. Signup

Endpoint: POST /users

Registers a new user account.

{
    "Username": "newUser",
    "Password": "password123"
    "Email": "newuser@example.com",
    "Birthday": "1990-01-01"
}

Sample Response:

    {
    "_id": "generatedId",
    "Username": "newUser",
    "Email": "newuser@example.com",
    "Birthday": "1990-01-01",
    "FavoriteMovies": []
    }

2. Login

Endpoint: POST /login

Logs in an existing user.

    {
        KEY: Username VALUE: existingUser
        KEY: Password VALUE: password123
    }

Sample Response: 

    {
            "user": {
                "_id": "generated id",
                "Username": "username",
                "Password": "hashed password",
                "Email": "newuser@example.com",
                "Birthday": "1990-01-01T00:00:00.000Z",
                "FavoriteMovies": [],
                "__v": 0
            },
        "token:" "generatedToken"
    }

3. View Movies

Endpoint: GET /movies

Shows all movies and information within the database.

        "Genre": "Genre Name",

        "ImagePath": "path/to/image",
        "Featured": true/false
    },
    {
    "Title": "Movie 2",

    "Description": "Description of Movie 2",
    "Genre": "Genre Name",
    "Director": "Director Name",
    "ImagePath": "path/to/image",
    "Featured": true/false
}

5. Add a Movie to Favorites

Endpoint: POST /users/{Username}/favorites/{MovieTitleToAdd}

This endpoint allows users to add a movie to their list of favorite movies.

      {
        "Favorites": ["Movie Title 1", "Movie Title 2", ...]
      }

6. View Favorite Movies

Endpoint: GET /users/{Username}/favorites

Returns a list of favorite movies for a specific user.

        {
            Title: Movie 1,
            id: generatedId1,
            Description: Description of Movie 1,
            Genre: {
                Name: Genre name of movie 1,
                Descritpion: Desription of Genre
            },
            Director: {
                Name" Director of movie 1,
                Bio: Biography of director
            }
        }
        ...etc

7. Remove Movie from Favorites

Endpoint: DELETE /users/{Username}/favorites/{MovieTitle}

Removes a movie from the list of favorite movies for a user.

      {
        "Favorites": ["Movie Title 1", "Movie Title 2", ...]
    }

8. Update Profile Data

Endpoint: PUT /users/{Username}

Allows a user to update their profile information.

      {
        "Username": "NewUsername",
        "Password": "NewPassword",
        "Email": "newemail@email.com",
        "Birthday": "YYYY-MM-DD"
      }

Sample Response:Based on what you changed/updated

    {
        "Username": "NewUsername",
    "Email": "newemail@example.com",
    "Birthday": "YYYY-MM-DD",
    "FavoriteMovies": [...]
    ...etc.
    }
