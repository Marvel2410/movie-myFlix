Favorite Movies App

Building a server-side component of a "movies" web aplication. The Web appilcation will provide users with access to information about some of my favorite movies, directors, and plots. Users will be able to sign up, update their personal information, and create a list of their own favorite movies.

API Endpoints

1. Signup

Endpoint: POST /users

Registers a new user account

Example Request Body:

{
    "Username": "newUser",
    "Password": "password123"
    "Email": "newuser@example.com",
    "Birthday": "1990-01-01"
}

Sample Response:

    {
    "_id": "generatedId",
    "Username": "newUser",
    "Email": "newuser@example.com",
    "Birthday": "1990-01-01",
    "FavoriteMovies": []
    }

2. Login

Endpoing: POST /login

Logs in an existing user.

Example Request Into Params:

    {
        KEY: Username VALUE: existingUser
        KEY: Password VALUE: password123
    }

Sample Response: 

    {
            "user": {
                "_id": "generated id",
                "Username": "username",
                "Password": "hashed password",
                "Email": "newuser@example.com",
                "Birthday": "1990-01-01T00:00:00.000Z",
                "FavoriteMovies": [],
                "__v": 0
            },
        "token:" "generatedToken"
    }

5. Add a movie to your favorites

Endpoint POST /users/{Username}/favorites/MovieTitletoaddtofavorites

This endpoint allows users to add a movie to their list of favorites movies

Example Request: 

      {
        /favorites/movietitletoadd
      }

6. View Favorite Movies

Endpoint: GET /users/{Username}/favorites

Returns a list of favorite movies for a specific user.

Sample Response:

        {

            Description: Description of Movie 1,
            Genre: {
                Name: Genre name of movie 1,

7. Remove movie from favorites

Enpoint: DELETE /users/Username/favorites/movietitle

Removes a movie from the list of favorite movies for a user.

Sample Response:

{ message: Movie 'title' removed from favorites sucessfully}
