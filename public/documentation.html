Favorite Movies of All Time

Building a server-side component of a "movies" web aplication. The Web appilcation will provide users with access to information about some of my favorite movies, directors, and plots. Users will be able to sign up, update their personal information, and create a list of their own favorite movies.

2. Return data about a single movie by title to the user

Title of the movie, description, genre, director, image URL, year it came out, etc.

Endpoint: GET /movies/Movie1

Sample Response: 

            {
                "title": "Movie 1",
                "description": "Hero saves the day",
                "genre": "Action, Adventure, Sci-Fi",
                "director": "Jane Doe",
                "imageURL":"#"
                "featured": "true/false"
            }

3. Allow new users to Register

Endpoint: POST /users

This enpoint allows new users to register and create an account

4. Allow users to add their user information (username)

Endpoint PUT /users/{userId}

This endpoint allows users to update their username and information

5. Allow users to add a movie to their list of favorites

Endpoint POST /users/{userId}/favorites

This endpoint allows users to add a movie to their list of favorites movies

6. Allow users to Remove a movie

Endpoint DELETE /users/{userId}/favorites/{movieId}

Allows users to remove a movie from their list of favorites

7. Allow existing user to deregister

Endpoint DELETE /users/{userId}

This endpoint allows for existing users to deregister or delete their account
